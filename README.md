# Torrent-app

Build a simple CLI torrent-app for P2P file sharing 

## Run project 
```sh
python main.py
```

## Virtual environment setup
### Create venv
```sh
python -m venv venv
```
### Active for window
```sh
python -m venv venv
```
### Active for Linux/MacOS
```sh
source venv/bin/activate
```
### Install dependencies
```sh
pip install -r requirements.txt
```
